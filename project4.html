Project Title: Image Classification Using CNNs

This code performs time series forecasting using a Linear Regression model. It consists of data normalization, pattern extraction, model training, forecasting, and evaluation of the model's performance using RMSE and MAE metrics.

Purpose

The code aims to forecast future values of a time series dataset using a linear regression model. It involves several steps including data preprocessing (normalization), pattern extraction, model training, forecasting, and evaluation.

Functions

This function extracts patterns from the time series data using a sliding window approach.
Parameters: TSeries: The time series data. n_inputs: Number of lagged observations to consider. h: The horizon for the prediction.
Returns: X: Input patterns. y: Corresponding outputs.
  • minmaxNorm
    • This function normalizes the data using Min-Max normalization.
    Parameters: originalData: The original time series data. lenTrainValidation: Length of the training and validation set.
    Returns: normalizedData: The normalized data.
  • minmaxDeNorm
    • This function denormalizes the forecasted data back to its original scale.
    Parameters: originalData: The original time series data. forecastedData: The forecasted data. lenTrainValidation: Length of the training and validation set. Returns: denormalizedData: The denormalized data.
  • findRMSE
    • This function calculates the Root Mean Square Error (RMSE) for training and testing sets. Parameters: Timeseries_Data: The original time series data. forecasted_value: The forecasted values. lenTrainValidation: Length of the training and validation set. Returns: trainRMSE: RMSE for the training and validation set. testRMSE: RMSE for the test set.
  • findMAE
    • This function calculates the Mean Absolute Error (MAE) for training and testing sets. Parameters: Timeseries_Data: The original time series data. forecasted_value: The forecasted values. lenTrainValidation: Length of the training and validation set. Returns: trainMAE: MAE for the training and validation set. testMAE: MAE for the test set.
  • Find_Fitness
    • This function trains the model and makes predictions.
    Parameters:
    x: Input patterns.
    y: Corresponding outputs.
    lenValid: Length of the validation set.
    lenTest: Length of the test set.
    model: The machine learning model to be used.
    Returns: yhatNorm: Normalized predictions.

    Data Loading

    Reads a time series dataset (Lynx.csv).

    Normalization

    Normalizes the data using minmaxNorm.

    Pattern Extraction

    Converts the time series into input (X) and output (y) patterns using get_Patterns.

    Model Training and Evaluation

    Uses a LinearRegression model.
    Iteratively trains the model, forecasts future values,
    evaluates performance using RMSE and MAE.
    Stores accuracy metrics (Accuracy) and forecasted values (Forecasts)
    in Excel files (LinearRegression_Accuracy.xlsx and LinearRegression_Forecasts.xlsx).
